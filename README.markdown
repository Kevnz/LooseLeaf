# LooseLeaf

JSON based blog engine running on [node.js][] and [express][].

[node.js]: http://nodejs.org/
[express]: http://expressjs.com/

## Installation

	$ npm install looseleaf

## Getting Started

	$ looseleaf yourblog
	$ node yourblog/app.js
