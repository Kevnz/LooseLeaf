# LooseLeaf

Lightweight blog engine running on [node.js][] and [express][].

	$ npm install -g looseleaf
	$ looseleaf yourblog
	$ node ./yourblog/app.js
	"yourblog" server listening on port 3000

[node.js]: http://nodejs.org/
[express]: http://expressjs.com/
